Valid Personal Identification Code
This program validates a Finnish Personal Identity Codes (PIC)
Finnish PICs follow the format ddmmyyXyyyz, where:
  • ddmmyy contains a valid date of birth,
  • X is the marker for century, which is either + (1800s), - (1900s) or A (2000s).
  • yyy is the personal identifier which can be any number between 002 and 899 (odd for men, even for women), and
  • z is a control character.

The control character is calculated by taking the nine-digit number created by the date of birth and the personal identifier, dividing this by 31, and multiplying the series of numbers remaining after the decimal place by 31. We then take the result and select the character at the index specified by the number from the string 0123456789ABCDEFHJKLMNPRSTUVWXY. For example, if the number was 12, the control character would be C

Example:

230827906/31 = 7446061.48387097

0.48387097 * 31 = 15

The index is 15 therefore the control character is F.

Here are some valid PICs: 230827-906F, 120488+246L, 310823A9877.
